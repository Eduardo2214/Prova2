Introdução

Esse website mostra alguns conteudos das atividades HTML realisadas em sala do curso de desenvolvimento front-end do 3° periodo de engenharia de software. O site é composto por 4 páginas interligadas contendo informações do conhecimento sobre HTML adiquirido durante as aulas.

Os conteúdos de cada página são:

  • Página 1: É a pagina de conteúdo que reúne algumas das informações adiquiridas sobre html
  • Página 2: É a galeria que contém as imagens dos projetos realizados mas dessa vez incluindo o CSS
  • Página 3: É a página de contato onde tem um formulário para ser preenchido com nome, E-mail e mensagem
