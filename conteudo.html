Conhecimentos HTML

Listas ordenadas

Em HTML, estudei sobre listas ordenadas, não ordenadas. As listas ordenadas (ol) são usadas para itens que têm uma ordem específica, enquanto as listas não ordenadas (ul) são usadas para itens que não têm uma ordem específica

Titulos e subtitulos

Em HTML, títulos e subtítulos são definidos pelas tags h1 a h6, onde h1 é o título principal e h6 o menor subtítulo. A tag h1 representa o título mais importante e as demais tags h2 a h6 indicam subtítulos de diferentes níveis, com ordem hierárquica decrescente de importância.

Links

Em HTML, links, são elementos que permitem a navegação entre diferentes páginas web ou recursos online. Eles são criados usando a tag a (âncora) e o atributo href especifica o destino do link.
